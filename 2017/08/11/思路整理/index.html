思路整理及学习计划

发表于 2017-08-11

总的来说，给自己定下一个清晰的目标：两年之后你要成为什么样的人。首先你必须是一个卓越的工艺，其次巩固并且丰富你的理论知识，成为具有研发实力的工程师。

首先说说产线

目前阶段是熟悉产线的阶段，各个工段的系统和设备可以通过阅读相关资料和现场观察来完成。尤其是前区混料、挤出和铸片系统是对隔膜厚度分布和稳定性影响极为重要的部分。当然双向拉伸和萃取系统直接决定隔膜的各项性能，对其设备和辅助系统也必须熟悉，如温控单元或者模温机位置及原理、TD和萃取的风机有何作用、萃取展平锟和面板操作等；收卷和大分切部分，也是工艺的另一个重要战地，收卷和大分切操作，主要是面板和穿模操作包括，各按键的含义。除此之外，也应当对测厚仪和瑕疵仪器的界面操作有一定的了解。到此，就已经对整条产线有一定程度的认识了，但是想要获得更广阔的发展就应当对质检和涂布有一定了解，在时间充裕的情况下可以去了解了解。

说完产线，就得回到主题–工艺调整。我将其分为三个细分主题。

一、瑕疵的本质、出现原因和调整方法，目前遇到的瑕疵主要是：

1. 厚度相关的厚薄带、横纹、暗纹、积料，
主要调节方法是改变接触锟间隙、机架位置
或者调节下料（可能与G/P波动、下料的频率、失重秤密封性有关）；
2. 黑点、积炭：原因不明，继续学习；
3. 白点、白斑：前区锟面异物；
4. 水纹、油点：萃取部分出现问题，可调节展平锟、盖子和速比。

二、厚度及性能调节

- 厚度的调节主要通过螺栓进行，此外，改变接触锟间隙或调节下料也对改变厚度有明显影响；
- 性能主要是孔隙率和透气，调节的位置无外乎2nd TD定型温度和1st TD拉伸温度；

三、大分切参数调节
目前还不熟悉，主要的调节手段为张力和接触锟压力调节。

再聊聊研发能力的培养

目前提升能力的主要手段是阅读相关文献资料：
跟薄膜制造相关的书、锂电隔膜相关的文献
（《双向拉伸塑料薄膜》、《聚乙烯锂离子电池隔膜的研究（陈俊钊）》）。
理论知识包括高分子材料、结晶学。
